## 🖥️ Montagem e Hardware

| Site | Descrição |
| :--- | :--- |
| [MEUPC.NET](https://meupc.net/build) | A melhor ferramenta brasileira para montar seu PC, conferir compatibilidade e comparar preços. |
| [PCPartPicker](https://pcpartpicker.com) | A plataforma global mais famosa para planejar *builds* e verificar preços internacionais. |
|[PCBuilder.net](https://pcbuilder.net/list) | Um construtor de PC intuitivo para criar listas de componentes e setups personalizados. |
| [Corsair PC Builder](https://www.corsair.com/br/pt/pc-builder) | Configurador oficial da Corsair para montar um setup focado na marca com compatibilidade garantida. |

## 🎮 Jogos e Passatempos

| Site | Descrição |
| :--- | :--- |
| [Wplace - Paint the World](https://wplace.live) | Pinte um mapa-múndi colaborativo em tempo real com outras pessoas. |
|[Infinite Craft](https://neal.fun/infinite-craft) | Um jogo de alquimia infinita onde você combina elementos para criar tudo. |
| [Wikigacha](https://wikigacha.com) | Um divertido jogo de navegador que transforma artigos aleatórios da Wikipédia em cartas colecionáveis em um sistema *gacha*. |
| [HaxBall](https://haxball.com) | Futebol de física simples e multiplayer direto no navegador. |
| [Slow Roads](https://slowroads.io) | Um simulador de direção infinita e relaxante para desestressar. |
| [2048](https://play2048.co) | O clássico puzzle numérico viciante de Gabriele Cirulli. |
|[Wordle](https://www.nytimes.com/games/wordle/index.html) | O famoso jogo diário de adivinhar palavras do NY Times. |
| [Little Alchemy (1 & 2)](https://littlealchemy2.com) | Misture elementos básicos para descobrir centenas de novos itens. |
|[Play Retro Games](https://gam.onl) | Uma enorme biblioteca de jogos clássicos de consoles antigos. |
| [CSS Clicker](https://lyra.horse/css-clicker) | Um jogo estilo *clicker* feito puramente com foco em CSS. |
| [Lose Lose](http://loselose.net) | 🔴 **PERIGO:** Um jogo que deleta um arquivo real do seu PC para cada inimigo morto. Jogue por sua conta e risco. |
